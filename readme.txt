Proyecto Web: Tienda Saludable

Este proyecto consiste en una mini tienda de productos de nutrición. 
Se compone de dos páginas HTML (index.html y contacto.html), una hoja de estilos CSS (styles.css) 
y este archivo de texto explicativo.

Herramientas y tecnologías utilizadas:
- HTML5 para la estructura del contenido.
- CSS3 para el diseño y la presentación.
- Google Fonts para las fuentes tipográficas.
- Maquetado utilizando divs con width y margin para la disposición de los elementos.

Las páginas contienen hiperenlaces entre sí, imágenes de productos, 
y un formulario de contacto básico.
